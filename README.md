# Funding_Shop_ver2
Spring Framework로 구현한 Funding Shop 리뉴얼 프로젝트 입니다.

## 📆제작 기간 
2019.12.23~2020.01.16

## 💻환경
- JAVA 1.8
- windows 10
- mariaDB
- STS3
- Eclipse
- Apache Tomact
